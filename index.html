ESP32 Web Flasher

ESP32-TOOLS-PRO-480x320-V2.0

Firmware multi-herramienta para ESP32 con pantalla 480x320, WiFi, BLE, IR, nRF24 y CC1101 sub-GHz para aprendizaje, diagnostico y laboratorio propio.

Flashear desde el navegador

Usa Chrome, Edge u otro navegador compatible con Web Serial. Conecta el ESP32 por USB, pulsa instalar y selecciona el puerto serial correcto.

Tu navegador no soporta Web Serial. Usa Chrome o Edge en escritorio. Web Serial esta bloqueado. Abre esta pagina con HTTPS o localhost.
  1. Conecta tu ESP32 por USB.
  2. Si tu placa lo requiere, manten presionado BOOT al iniciar el flasheo.
  3. Selecciona el puerto serial y espera a que termine.
  4. Reinicia el ESP32 despues de flashear.
